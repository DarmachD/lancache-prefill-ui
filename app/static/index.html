How it behaves

        

          
Start prefill creates a server-side job. Closing this page or shutting the laptop does not stop it.

          
Select games opens the normal SteamPrefill selector. The interactive console is only created when needed.

          
Keep CacheDeck LAN-only. Docker socket access gives the container powerful control over the host.
